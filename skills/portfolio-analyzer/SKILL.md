# Portfolio Analyzer - 投资组合分析

## 功能概述

管理投资组合，分析持仓收益，评估投资风险。

## 核心功能

### 1. 持仓管理

| 功能 | 说明 |
|------|------|
| 添加持仓 | 记录买入的股票、数量、成本价 |
| 更新持仓 | 修改持仓数量、成本价 |
| 删除持仓 | 卖出后移除持仓 |
| 查看持仓 | 显示当前所有持仓 |

### 2. 收益分析

| 指标 | 说明 |
|------|------|
| 持仓收益率 | (现价 - 成本价) / 成本价 |
| 总收益率 | 组合总盈亏 / 总成本 |
| 年化收益率 | 按年化计算的收益率 |
| 超额收益 | 相对基准(沪深300)的超额收益 |

### 3. 风险评估

| 指标 | 说明 |
|------|------|
| 夏普比率 | 风险调整后收益，越高越好 |
| 最大回撤 | 历史最大下跌幅度 |
| 波动率 | 收益率的标准差 |
| VaR | 95%置信度下的最大损失 |

### 4. 资产配置

| 分析 | 说明 |
|------|------|
| 行业分布 | 按行业分类统计持仓 |
| 集中度 | 单一股票/行业占比 |
| 风格分析 | 大盘/中盘/小盘 |

## 使用方法

### 管理持仓

```
@mk 添加持仓: 600519 贵州茅台 100股 成本1500元
@mk 更新持仓: 000858 五粮液 200股
@mk 删除持仓: 600519
```

### 查看组合

```
@mk 查看我的持仓
@mk 组合收益情况
@mk 风险分析
```

### 性能分析

```
@mk 本月收益多少
@mk 组合夏普比率
@mk 资产配置分析
```

## 数据存储

持仓数据保存在: `~/.openclaw/data/portfolio.json`

```json
{
  "positions": [
    {
      "symbol": "600519",
      "name": "贵州茅台",
      "shares": 100,
      "cost": 150000,
      "costPrice": 1500,
      "addedAt": "2026-01-15"
    }
  ]
}
```

## 注意事项

1. 成本价需要手动更新(不支持自动同步)
2. 收益为理论收益，未考虑交易费用
3. 历史数据需要额外订阅
