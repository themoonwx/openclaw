# Market Discovery - 市场发现

## 功能概述

发现市场中的投资机会，追踪新兴行业和趋势变化。

## 核心功能

### 1. 投资机会

| 类型 | 说明 |
|------|------|
| 低估机会 | PE/PB 低于行业平均 |
| 业绩拐点 | 业绩增速由负转正 |
| 政策利好 | 受益于政策支持 |
| 技术突破 | 关键技术进展 |

### 2. 趋势变化

| 类型 | 说明 |
|------|------|
| 均线多头 | 短期均线向上突破 |
| 放量突破 | 成交量放大突破关键位 |
| 趋势反转 | 下跌趋势转为上涨 |
| 板块轮动 | 资金从一板块流向另一板块 |

### 3. 新兴行业

| 类型 | 说明 |
|------|------|
| 新兴赛道 | AI、人形机器人、脑机接口等 |
| 技术迭代 | 新一代技术带来的机会 |
| 消费升级 | 新兴消费领域 |
| 国产替代 | 被卡脖子的关键技术 |

### 4. 异动提醒

| 类型 | 说明 |
|------|------|
| 涨停异动 | 首次涨停的股票 |
| 资金异动 | 大单净流入/净流出 |
| 消息异动 | 重大利好/利空消息 |

## 使用方法

```
@mk 帮我发现低估的股票
```

### 新兴行业


### 异动监控

```

```

## 分析方法

1. **数据挖掘**: 从财经新闻、公告、研报中提取机会
2. **趋势识别**: 技术面 + 资金面 + 政策面
3. **模式匹配**: 历史相似走势模式识别
4. **信号验证**: 多维度确认机会可靠性

## 注意事项

1. 机会发现基于历史数据和算法，存在不确定性
2. 建议结合基本面分析做出投资决策
3. 市场有风险，投资需谨慎
