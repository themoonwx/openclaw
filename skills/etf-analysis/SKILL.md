# ETF Analysis - ETF 分析工具

## 功能概述

分析 ETF 产品，提供筛选、推荐、溢价率分析等功能。

## 核心功能

### 1. ETF 筛选

| 类型 | 说明 |
|------|------|
| 宽基 ETF | 沪深300、中证500、创业板等 |
| 行业 ETF | 新能源车、半导体、医药等 |
| 策略 ETF | 红利、低波动、成长等 |
| 跨境 ETF | 港股通、美股 ETF |

### 2. 热门 ETF

| 指标 | 说明 |
|------|------|
| 涨幅榜 | 最近涨得最好的 ETF |
| 换手榜 | 交易最活跃的 ETF |
| 资金流入 | 最近净流入最多的 ETF |

### 3. 溢价率分析

| 指标 | 说明 |
|------|------|
| 溢价率 | 交易价格 vs 净值 |
| 折价率 | 交易价格 < 净值 |
| 建议 | 买入/卖出/持有 |

### 4. 组合推荐

| 策略 | 说明 |
|------|------|
| 稳健型 | 宽基 + 债券 |
| 平衡型 | 宽基 + 行业 |
| 激进型 | 行业 + 主题 |
