# Sentiment Analysis - 市场情绪分析

## 功能概述

分析市场整体情绪和个股情绪，识别市场热点和风险。

## 情绪类型

### 1. 整体市场情绪

| 情绪 | 说明 | 信号 |
|------|------|------|
| 极度乐观 | 市场过热 | 风险信号 |
| 乐观 | 市场偏多 | 继续持有 |
| 中性 | 观望为主 | 谨慎操作 |
| 悲观 | 市场偏空 | 注意风险 |
| 极度悲观 | 市场冰点 | 机会信号 |

### 2. 情绪指标

| 指标 | 说明 |
|------|------|
| 成交量 | 市场活跃度 |
| 融资融券 | 杠杆情绪 |
| 新开户数 | 散户情绪 |
| 股民情绪指数 | 综合指标 |

### 3. 个股情绪

| 指标 | 说明 |
|------|------|
| 新闻热度 | 新闻数量和情感倾向 |
| 社交讨论 | 微博、雪球讨论量 |
| 研报评级 | 分析师推荐 |
| 资金流向 | 主力资金进出 |

## 使用方法

### 市场情绪

```
@mk 市场情绪如何
@mk 今日多空对比
@mk A股情绪指标
```

### 个股情绪

```
@mk 贵州茅台情绪分析
@mk 比亚迪热度怎么样
@mk 宁德时代最近讨论多吗
```

### 行业情绪

```
@mk 新能源车行业情绪
@mk 半导体板块情绪
```

## 情绪数据来源

- 新闻: 东方财富、同花顺
- 社交: 微博、雪球
- 研报: 券商研报
- 资金: 主力资金流向

## 分析方法

### 1. 自然语言处理
- 新闻标题情感分析
- 文本关键词提取
- 情绪词汇匹配

### 2. 量化指标
- 成交量异常检测
- 资金流向分析
- 涨跌停统计

### 3. 舆情监控
- 新闻数量统计
- 社交媒体热度
- 搜索指数

## 注意事项

1. 情绪分析仅供参考，不构成投资建议
2. 历史表现不代表未来
3. 请结合基本面和技术面分析
