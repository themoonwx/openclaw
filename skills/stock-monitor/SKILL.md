# Stock Monitor - 个股监控

## 功能概述

实时监控关注的股票价格异动，支持多种监控模式和告警通知。

## 监控类型

### 1. 价格监控

| 类型 | 说明 |
|------|------|
| 涨跌幅监控 | 涨/跌超设定阈值 |
| 价格突破 | 突破设定价格 |
| 均线金叉/死叉 | 技术指标信号 |

### 2. 成交量监控

| 类型 | 说明 |
|------|------|
| 放量/缩量 | 成交量较昨日放大/缩小 |
| 成交量突增 | 短时间内成交量急剧增加 |
| 活跃度 | 换手率异常 |

### 3. 基本面监控

| 类型 | 说明 |
|------|------|
| 业绩预告 | 发布业绩预告/快报 |
| 限售股解禁 | 限售股解禁提醒 |
| 大宗交易 | 大宗交易提醒 |

### 4. 消息面监控

| 类型 | 说明 |
|------|------|
| 公告监控 | 重要公告发布 |
| 新闻监控 | 相关新闻报道 |
| 研报监控 | 券商研报发布 |

## 使用方法

### 添加自选股

```
@market 关注 600519 贵州茅台
@market 添加自选 000858 五粮液
@market 监控 比亚迪 涨跌幅>5%
```

### 查看监控

```
@market 查看自选股
@market 我的监控列表
@market 今日异动
```

### 设置告警

```
@market 设置贵州茅台涨超 1800 元提醒
@market 设置宁德时代跌超 5% 提醒
@market 取消比亚迪的监控
```

## 通知渠道

- 企业微信 (wecom) - 默认
- 飞书 (feishu)
- 钉钉 (dingtalk)

## 数据来源

- 实时行情: 东方财富/同花顺
- 收盘数据: 每日更新

## 注意事项

1. 监控频率: 5分钟刷新一次
2. 自选股上限: 50 只
3. 告警记录保留: 7 天
