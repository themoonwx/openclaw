# Stock Screener - 股票筛选器

## 功能概述

根据多维度条件筛选符合条件的股票。

## 筛选维度

### 基本面指标

| 指标 | 说明 | 常用筛选 |
|------|------|----------|
| 市值 | 股票总市值 | >100亿, >500亿 |
| PE (市盈率) | 价格/每股收益 | <20, <30 |
| PB (市净率) | 价格/每股净资产 | <3, <5 |
| ROE (净资产收益率) | 净利润/净资产 | >10%, >15% |
| 毛利率 | 毛利/营业收入 | >30% |
| 净利润增速 | 净利润同比增长 | >20% |

### 技术面指标

| 指标 | 说明 | 常用筛选 |
|------|------|----------|
| 涨跌幅 | 日/周/月涨跌幅 | >5%, >10% |
| 成交量 | 日成交量 | >1亿 |
| 换手率 | 日换手率 | >5% |
| 股价 | 股票价格 | <50元 |

### 行业/概念

| 类型 | 说明 |
|------|------|
| 行业 | 农林牧渔、医药、银行等 |
| 概念 | 新能源车、人工智能、芯片等 |

## 使用方法

### 示例1: 筛选低估值股票

```
@mk 筛选 PE<20, ROE>15% 的股票
```

### 示例2: 筛选热门概念

```
@mk 筛选新能源车概念、市值>500亿的股票
```

### 示例3: 筛选技术突破

```
@mk 筛选最近5天涨幅>10%且成交量放大的股票
```

## 注意事项

1. 筛选结果最多返回 50 只股票
2. A 股数据主要来自东方财富
3. 财务数据可能有延迟，请以实际公告为准
