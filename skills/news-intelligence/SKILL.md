# News Intelligence - 新闻情报

## 功能概述

聚合市场新闻、个股公告、研报等情报信息，支持多维度的新闻筛选和关键词监控。

## 新闻类型

### 1. 宏观新闻

| 类型 | 说明 |
|------|------|
| 政策 | 货币政策、财政政策、产业政策 |
| 经济 | GDP、CPI、PPI 等经济数据 |
| 国际 | 美联储、欧央行等国际动态 |

### 2. 行业新闻

| 类型 | 说明 |
|------|------|
| 新能源 | 电动车、光伏、储能 |
| 科技 | 芯片、人工智能、软件 |
| 医药 | 创新药、医疗器械 |
| 消费 | 白酒、食品饮料 |

### 3. 个股新闻

| 类型 | 说明 |
|------|------|
| 业绩 | 业绩预告、业绩快报、年报 |
| 重组 | 并购重组、定增配股 |
| 减持/增持 | 大股东增减持 |
| 诉讼 | 重大诉讼仲裁 |

### 4. 公告

| 类型 | 说明 |
|------|------|
| 定期报告 | 年报、半年报、季报 |
| 临时公告 | 重要合同、担保、处罚 |
| 融资公告 | 发行股份购买资产 |

## 使用方法

### 获取新闻

```
@mk 今日新闻
@mk 市场快讯
@mk 有什么重大新闻
```

### 搜索新闻

```
@mk 搜索 比亚迪 相关新闻
@mk 搜索 新能源车 政策
@mk 宁德时代的新闻
```

### 关键词监控

```
@mk 设置关键词监控: 人工智能
@mk 监控: 芯片 半导体
@mk 查看监控关键词
```

## 输出格式

```
## 📰 今日要闻

### 🔴 重要
1. **央行：保持流动性合理充裕**
   - 来源: 新华网 | 时间: 10:30
   - 摘要: 央行表示将保持流动性合理充裕...

2. **工信部：推动新能源汽车高质量发展**
   - 来源: 工信部网站 | 时间: 09:45
   - 摘要: 工信部将推动新能源汽车产业...

### 🟡 一般
...

## 📋 个股新闻

| 股票 | 新闻标题 | 时间 |
|------|----------|------|
| 贵州茅台 | 业绩预增 15% | 15:30 |
| 比亚迪 | 销量突破 50 万辆 | 14:20 |
```

## 数据来源

- 财经网站: 东方财富、同花顺、新浪财经
- 官方发布: 证监会、交易所、公司公告
- 研报: 券商研报摘要

## 注意事项

1. 新闻更新频率: 30 分钟
2. 历史新闻保留: 7 天
3. 研报仅提供摘要，完整研报需付费
